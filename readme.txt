

1. Download XAMPP

2. Download the zip file 

3. Extract the file and copy "Payroll_Management_System()" folder.

4. Run Apache and Mysql server in Xampp

5. Paste inside htdocs directory, where you install xammp (my path is C:\xampp\htdocs\Payroll_Management_System()).

6. Open PHPMyAdmin (http://localhost/phpmyadmin)

7. Create a database with name payroll

8. Import payroll.sql file(given inside the zip package in .SQL file)

9. Run the script http://localhost/Payroll_Management_System()
